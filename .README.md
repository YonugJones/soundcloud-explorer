A TypeScript + React + Express app that connects to the SoundCloud API via OAuth2.

## Features

- Login with SoundCloud
- View your SoundCloud profile
- Search tracks using the SoundCloud API
- Built with Vite, TailwindCSS, and TypeScript

## Setup

1. `cd backend && npm install`
2. `cd frontend && npm install`
3. Create `.env` files in backend (see `.env.example`)
4. Run backend: `npm start`
5. Run frontend: `npm run dev`
